# 🧬 HerediCare

**HerediCare** is a specialized web-based management system developed as a research prototype within the **College of Informatics and Computing Sciences** at **Batangas State University**.

## 📝 Project Description
HerediCare is designed to streamline the management of medical or hereditary data through a secure and automated digital interface. As a **Computer Science student researcher**, I developed this system to address the need for structured data handling in specialized healthcare contexts.

A core component of this project is its **Risk Assessment** module. This feature is specifically designed to analyze hereditary patterns, allowing users to become more aware of potential health risks within their family. By providing these insights, the system aims to bridge the gap between data collection and proactive health awareness.

The project focuses on the intersection of **Software Development** and **Socially Relevant Design**, ensuring that sensitive information is managed with both technical efficiency and user-centric security. It serves as a demonstration of implementing modern web frameworks to solve real-world informatics challenges, drawing on my experience with hardware and software prototypes.

## 🚀 Key Features
*   **Family Risk Assessment:** Tools to identify and visualize potential hereditary health risks.
*   **Secure Authentication:** Multi-level user login and session management to ensure data privacy.
*   **Automated Record Management:** Backend logic built for high-speed data retrieval and storage.
*   **Dynamic Analytics Dashboard:** A centralized interface for visualizing records and system statistics.
*   **Responsive Research Interface:** A clean, professional UI optimized for research environments.

## 🛠️ Technical Stack
*   **Backend:** Python (Flask Framework)
*   **Database:** SQLite3
*   **Frontend:** HTML, CSS, JavaScript
*   **Development Tools:** Git Bash, VS Code
